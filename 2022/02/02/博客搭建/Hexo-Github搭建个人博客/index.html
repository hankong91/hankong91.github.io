Hexo+Github搭建个人博客的步骤，及可能遇到的坑

安装Node.js

  • Hexo需要Node.js支持
  • Node.js下载地址：nodejs.org
  • 安装完以后会有两个组件，Node.js + npm包管理器

安装cnpm，淘宝镜像源

  • 借助npm安装Hexo博客框架，但由于国内镜像源速度慢，可以利用npm安装一个cnpm，淘宝的源
